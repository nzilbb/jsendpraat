Installation of Praat Integration

To integrate your browser with Praat, you need to take the following three steps:

1. Install Praat

If you don't already have Praat, download and install Praat from the Praat website:
praat.org

2. Install the Praat Integration browser extension

You have already done this; the browser extension is what is showing you these instructions.

3. Install the Praat Messaging Host

This is the component that sends messages from the browser extension to Praat. This uses Java, so if you don't already have a recent version of Java, download and install it now:
www.java.com

To install the Praat Messaging Host:

  1. Download and save the following file:
    install-jsendpraat.jar
  2. Click on the file to open it.
    You will see a window open called Install Praat Messaging Host.
  3. Click the Install button.

On OS X systems, if you see a message saying install-jsendpraat.jar can't be opened because it is from an unidentified developer you need to take the following additional steps:

  1. Click OK on the message.
  2. Go to you Mac's System Preferences and click Security and Privacy.
  3. Click the Open Anyway button, and then click Open.

Once the Praat Messaging Host is installed, you're free to delete the install-jsendpraat.jar file that you downloaded, and close this tab.
